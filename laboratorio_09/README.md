# Laboratorio 09 - Método de Euler

## Compilar código C++:
g++ main.cpp -o laboratorio09 -std=c++11

## Ejecutar programa:
./laboratorio09

## Generar gráficas:
python3 graficar.py (matplotlibcpp no esta disponible ni se puede instalar en la maquina virtual de Ubuntu)

## Archivos entregados:
- main.cpp : Código fuente completo
- laboratorio09 : Programa ejecutable
- output.txt : Resultados numéricos y error máximo
- datos_graficas.txt : Datos utilizados para graficar
- graficar.py : Código para generar las imágenes
- solucion_euler.png : Gráfica comparativa Euler vs Exacta
- error_local.png : Gráfica del error absoluto
- README.md : Instrucciones
